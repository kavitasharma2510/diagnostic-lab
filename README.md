# diagnostic-lab

Tyagi Pathology — Diagnostic Laboratory Management (MERN + MongoDB + Prisma).

## Local development

```bash
npm install
cp backend/.env.example backend/.env   # set MONGO_URI
npm run dev
```

- UI: http://localhost:3000  
- API: http://localhost:5000
